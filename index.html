Политика конфиденциальности

Приложение: 3D Printing Calculator
Дата обновления: 31.03.2026

Мы уважаем вашу конфиденциальность. Эта политика описывает, какие данные обрабатываются в приложении 3D Printing Calculator и для каких целей.

1. Какие данные обрабатываются

  • Данные, которые пользователь вводит вручную (модели, расчеты, заказы, клиенты, настройки).
  • Данные, связанные с файлами, создаваемыми пользователем (экспорт документов, резервные копии).
  • Технические данные устройства, необходимые для работы функций приложения и лицензирования Pro.

2. Цели обработки

  • Корректная работа функций приложения.
  • Сохранение пользовательских настроек и истории.
  • Экспорт документов и резервное копирование.
  • Проверка и активация лицензии Pro.

3. Передача данных третьим лицам

Мы не продаем персональные данные пользователей. Передача возможна только в рамках работы инфраструктуры приложения и в случаях, предусмотренных законом.

4. Хранение данных

Основные пользовательские данные хранятся локально на устройстве пользователя. Пользователь самостоятельно управляет резервными копиями и экспортированными файлами.

5. Удаление данных

Вы можете удалить данные через функции приложения и/или удалить приложение с устройства.
